looking for a right owner
search > right owners
fuzzy search fuzzy searches carry out a approximate search on last name, first name, main id, name number or any other identifier. The results are displayed in the results grid, sorted by relevance. Fuzzy searches are case insensitive, and can return results that don't exaclty match entered values.
exact search exact search is exact "begins" for identifiers, and exact "contains" for names. It is case insensitive. Exact searches are useful for example to search by CMO of affiliation or by type.

Exact searches can be saved as favorites and relaunched from the favorites menu. Favorites are sorted alphabetically in the favorites menu.
search shared By default, fuzzy or exact searches are carried out on the local database. By selecting "search shared", the results found on the shared database are displayed. In order to use such results, they should be synchronized first (see synchronization)
 
  Search can be launched with the return key.

Other fields can be used in the exact search for finding a creator/performer. Such fields can be the name, first name, main id, identifier, affiliated CMO, registration dates, date of birth, country of birth, citizenship, tags or combination of these fields
